ImportX
=======

*Extend the ability of Wolfram Language to include the ability to import external resources.*

## Installation

## Support Formats

| Format | Extension   |
|--------|-------------|
| TOML   | toml        |
| YAML   | yaml, yml   |
| JSON5  | json5, json |

### TOML

```wolfram
toml = PacletObject["ImportX"]["AssetLocation", "example_toml"];
Import[toml, {"TOML", "Text"}]
Import[toml, "TOML"]
```

### YAML

```wolfram
toml = PacletObject["ImportX"]["AssetLocation", "example_toml"];
Import[toml, {"TOML", "Text"}]
Import[toml, "TOML"]
```
